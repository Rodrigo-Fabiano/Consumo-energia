O projeto da calculadora de energia,onde o usuário entra com os dados: equipamento, potência (watts) e o tempo de uso.
O sistema retorna após o cálculo de consumo mensal :(potencia * horasDia * 30) / 1000.
Passo 1 - Entre com o nome do equipamento.
Passo 2 - Informe a potência consumida informada pelo fabricante.
Passo 3 - Informe o tempo de uso diário.
Linguagem utilizada Python.
Nome do Projeto - Calculadora de Consumo Energético
